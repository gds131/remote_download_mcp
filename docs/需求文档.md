# 远程下载管理MCP服务需求文档（配置化版）

## 1. 项目概述

基于MCP协议的远程下载服务，支持用户通过外挂配置文件自定义多个下载器实例（如qb1、qb2、tr1等），部署于Docker。

---

## 2. 功能需求

### 2.1 核心功能

| 功能模块 | 描述 | 优先级 |
|---------|------|--------|
| **下载任务添加** | 接收下载地址、保存路径、下载器别名，提交下载任务 | P0 |
| **多下载器支持** | 通过配置文件支持任意数量的下载器实例 | P0 |
| **配置热加载** | 配置文件变更后自动生效（或重启后生效） | P1 |
| **健康检查** | 服务端和下载器连接状态检查 | P0 |

### 2.2 明确不开发的功能

- ❌ 状态查询
- ❌ 分类管理
- ❌ 通知机制
- ❌ 任务控制（暂停、恢复、删除）
- ❌ 代码中硬编码下载器类型限制

---

## 3. 配置文件规范

### 3.1 配置文件格式

**文件路径**: `/app/config/downloaders.yaml`（Docker外挂路径）

```yaml
downloaders:
  # qBittorrent 实例1
  qb1:
    type: qbittorrent
    url: http://192.168.10.200:8081
    username: test
    password: testadmin
    
  # qBittorrent 实例2（可能是不同服务器）
  qb2:
    type: qbittorrent
    url: http://192.168.10.224:8080/
    username: test
    password: testadmin
    
  # Transmission 实例1
  tr1:
    type: transmission
    url: http://192.168.10.222:8080
    username: test
    password: testadmin
    

# 可选：默认下载器
default_downloader: qb1
```

### 3.2 配置字段说明

| 字段 | 说明 | 必填 | 可选值 |
|------|------|------|--------|
| `downloaders` | 下载器配置字典 | 是 | - |
| `<alias>` | 用户自定义别名（如qb1/tr2） | 是 | 任意字符串 |
| `type` | 下载器类型 | 是 | `qbittorrent`, `transmission` |
| `url` | Web UI/RPC地址 | 是 | 完整URL |
| `username` | 用户名 | 是 | - |
| `password` | 密码 | 是 | - |
| `default_downloader` | 默认下载器别名 | 否 | 必须是存在的alias |

## 6. 核心代码

### 6.1 config.py

```python
import yaml
from pathlib import Path
from typing import Dict, Any

CONFIG_PATH = Path("/app/config/downloaders.yaml")

class DownloaderConfig:
    def __init__(self, alias: str, type_: str, url: str, username: str, password: str):
        self.alias = alias
        self.type = type_
        self.url = url
        self.username = username
        self.password = password

class ConfigManager:
    _instance = None
    _config: Dict[str, DownloaderConfig] = {}
    _default_alias: str = None
    
    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
            cls._instance._load_config()
        return cls._instance
    
    def _load_config(self):
        """从YAML文件加载配置"""
        if not CONFIG_PATH.exists():
            raise FileNotFoundError(f"配置文件不存在: {CONFIG_PATH}")
        
        with open(CONFIG_PATH, 'r', encoding='utf-8') as f:
            data = yaml.safe_load(f)
        
        for alias, cfg in data.get('downloaders', {}).items():
            self._config[alias] = DownloaderConfig(
                alias=alias,
                type_=cfg['type'],
                url=cfg['url'],
                username=cfg['username'],
                password=cfg['password']
            )
        
        self._default_alias = data.get('default_downloader')
    
    def get_downloader(self, alias: str) -> DownloaderConfig:
        if alias not in self._config:
            raise ValueError(f"未知的下载器别名: {alias}，可用别名: {list(self._config.keys())}")
        return self._config[alias]
    
    def list_aliases(self) -> list[str]:
        return list(self._config.keys())
    
    def get_default(self) -> str:
        return self._default_alias

settings = ConfigManager()
```

### 6.2 server.py

```python
from mcp.server.fastmcp import FastMCP
from src.config import settings
from src.downloaders.qbittorrent import QBittorrentClient
from src.downloaders.transmission import TransmissionClient

mcp = FastMCP("remote-download")

# 下载器实例缓存
_downloader_instances: dict[str, any] = {}

def get_downloader(alias: str):
    """根据别名获取或创建下载器实例"""
    if alias not in _downloader_instances:
        cfg = settings.get_downloader(alias)
        
        if cfg.type == "qbittorrent":
            _downloader_instances[alias] = QBittorrentClient(
                url=cfg.url,
                username=cfg.username,
                password=cfg.password
            )
        elif cfg.type == "transmission":
            _downloader_instances[alias] = TransmissionClient(
                url=cfg.url,
                username=cfg.username,
                password=cfg.password
            )
        else:
            raise ValueError(f"不支持的下载器类型: {cfg.type}")
    
    return _downloader_instances[alias]

@mcp.tool()
async def add_download(
    download_url: str,
    downloader_alias: str,
    save_path: str = "/downloads"
) -> dict:
    """提交下载任务到指定远端下载器"""
    downloader = get_downloader(downloader_alias)
    result = await downloader.add_torrent(
        url=download_url,
        save_path=save_path
    )
    return {
        "success": True,
        "message": "下载任务已提交",
        "downloader": downloader_alias,
        "save_path": save_path
    }

@mcp.tool()
async def list_downloaders() -> dict:
    """列出所有配置的下载器别名"""
    return {
        "aliases": settings.list_aliases(),
        "default": settings.get_default(),
        "count": len(settings.list_aliases())
    }

@mcp.get("/health")
async def health_check():
    """健康检查"""
    status = {}
    for alias in settings.list_aliases():
        try:
            dl = get_downloader(alias)
            await dl.ping()
            status[alias] = "connected"
        except Exception as e:
            status[alias] = f"disconnected: {str(e)}"
    
    return {
        "status": "ok" if any(s == "connected" for s in status.values()) else "degraded",
        "downloaders": status
    }

if __name__ == "__main__":
    mcp.run(transport="streamable-http", port=8000)
```

## 7. Docker配置

### 7.1 Dockerfile

```dockerfile
FROM python:3.11-slim

WORKDIR /app

# 安装依赖
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# 复制源码
COPY src/ ./src/

# 创建配置目录（外挂卷挂载点）
RUN mkdir -p /app/config

EXPOSE 8000

HEALTHCHECK --interval=30s --timeout=10s --retries=3 \
    CMD python -c "import urllib.request; urllib.request.urlopen('http://localhost:8000/health')"

CMD ["python", "-m", "src.server"]
```

### 7.2 docker-compose.yml

```yaml
version: '3.8'

services:
  mcp-download:
    build:
      context: .
      dockerfile: docker/Dockerfile
    image: remote-download-mcp:latest
    container_name: download-mcp
    ports:
      - "${MCP_PORT:-8000}:8000"
    volumes:
      # 外挂配置文件
      - ./config:/app/config:ro
    environment:
      - MCP_HOST=0.0.0.0
      - MCP_PORT=8000
    restart: unless-stopped
```

## 8. 部署步骤

```bash
# 1. 创建配置目录和文件
mkdir -p config
cat > config/downloaders.yaml << 'EOF'
downloaders:
  qb1:
    type: qbittorrent
    url: http://192.168.1.100:8080
    username: admin
    password: yourpassword
    
  tr1:
    type: transmission
    url: http://192.168.1.101:9091
    username: admin
    password: yourpassword

default_downloader: qb1
EOF

# 2. 启动服务
docker-compose up -d

# 3. 验证
curl http://localhost:8000/health
curl http://localhost:8000/list_downloaders
```
